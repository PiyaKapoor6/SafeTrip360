SafeTrip 360°

SafeTrip 360° is a travel assistant platform that helps users plan safe and enjoyable trips with real-time insights, personalized recommendations, and interactive maps.

Features
Personalized trip planning and itineraries
Real-time safety alerts and travel advisories
Interactive 360° maps for destination visualization

Tech Stack
Frontend: Next.js, Tailwind CSS, Typescript
Backend: Node.js
APIs: Map and Travel Safety APIs, Weather, Gemini, NASA EONET API Key

Installation
git clone https://github.com/PiyaKapoor6/SafeTrip360.git
cd safetrip360
npm install
npm run dev
Set environment variables in .env (API keys, DB URL)
Open http://localhost:3000 in your browser

Target Audience
Solo travelers, families, tourists seeking safe trips
